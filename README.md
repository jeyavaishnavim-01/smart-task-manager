# Smart Task Manager #

A sleek, responsive and modern Task Management application designed for high productivity. Built with a focus on **Glassmorphism** UI and smooth user experience.

**Live Demo**
https://jeyavaishnavim-01.github.io/smart-task-manager/

**Key Features**

1. Glassmorphism UI: Modern "frosted glass" aesthetic with smooth hover effects.
2. Dark Mode: Eye-friendly theme toggle for late-night productivity.
3. Persistence: Tasks are saved to `localStorage`, so they won't disappear on refresh.
4. Categorization: Organize tasks into Work, Personal, or Urgent buckets.
5. Responsive: Optimized for both Desktop (large screen) and Mobile views.

**Tech Stack**
1. HTML5: Semantic structure.
2. CSS3: Custom variables, Flexbox, and Grid.
3. JavaScript (ES6+): DOM manipulation and Local Storage API.
4. Lucide Icons: Premium vector iconography.
